## ğŸ¯ Design Philosophy

### Color Translation

Translating a racing livery to UI isn't just about copying colors. We consider:

1. **Emotional Impact** - Gulf feels calm and professional, Marlboro feels aggressive and bold
2. **Readability** - Text must remain readable across all themes
3. **Hierarchy** - Primary colors for CTAs, secondary for backgrounds, accents for highlights
4. **Pattern Integration** - Subtle stripe patterns honor the original designs without overwhelming

### Dark Mode First

All themes are dark mode by default because:
- Motorsport data is often viewed in dim environments (race events, late nights)
- Dark backgrounds reduce eye strain during long sessions
- Racing livery colors pop better against dark backgrounds
- It's more energy-efficient on OLED displays

### Accessibility

Each theme maintains:
- Minimum 4.5:1 contrast ratio for body text
- Minimum 3:1 contrast ratio for large text and UI elements
- Clear focus states for keyboard navigation
- Color-blind friendly position badge designs
